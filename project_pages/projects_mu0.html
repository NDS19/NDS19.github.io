    • Overview

      This project was completed as part of our end of 1st year project for Electronic Engineering at Imperial College London. It was based on the Digital Electronics and Computer Architecture module.
      We were tasked with designing a CPU with an ISA optimised for the following algorithms:

      • Calculate Fibonacci numbers using recursion,
      • Calculate pseudo-random integers with a linear congruential generator (LCG),
      • Traverse a linked list to find an item.

      In addition to designing the CPU, we also had to write a report discussing the aspects of our CPU such as the speed when completing the tasks and power consumption.

    • My Roles

      I mainly worked on optimising the CPU for the task of calculating Fibonacci numbers using recursion. This task introduced me to stack memory which uses the 'last in, first out' data structure which we were directed to use to implement this task.
