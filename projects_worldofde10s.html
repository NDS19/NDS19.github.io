    • Overview

      World of DE-10s is a 1v1 tank game developed as part of the 2nd year Information Processing module completed in Spring 2021. The project involves an IoT system where 2 FPGAs, which act as controllers, are connected to PCs which act as nodes which communicate with a TCP server hosted on an AWS client. This server then handles communication between the player nodes and a 3rd PC which is running the game in unity, and sends and receives the required information between these 3 nodes to run the game.
